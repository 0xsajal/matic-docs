#
 Polygon 使用方式 {#how-polygon-works}

Polygon 是一个区块链应用平台，可提供混合权益证明和 Plasma 支持的侧链。

Polygon 具有三层架构：

1. 以太坊上的质押和 Plasma 智能合约
2. Heimdall（权益证明层）
3. Bor（区块生产者层）
