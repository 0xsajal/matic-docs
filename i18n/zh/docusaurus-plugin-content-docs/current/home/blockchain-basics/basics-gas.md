# 什么是燃料？ {#what-is-gas}

“燃料”是启用智能合约的区块链网络中使用的一种特殊单位的名称。它测量执行一个操作或一组操作所需的“工作量”：例如，要计算一个 Keccak256 加密计算的哈希值，每进行一次哈希运算将花费 30 个燃料，加上对每 256 比特数据进行哈希运算的成本（6 个燃料）。每次可以通过交易或平台上的合约执行的操作都需要一定数量的气体，而需要更多的计算资源，而需要更多的计算资源，而需要更少计算资源。
